# AI Engineering Repository

A collection of AI-powered projects and chatbot applications covering diverse domains such as law, travel, education, health, and personalized recommendations.

## Repository Structure

### 1. [Bilingual-RAG-Chatbot-on-Constitution-of-Kenya](https://github.com/Lord-LLM/AI_Engineering/tree/main/Bilingual-RAG-Chatbot-on-Constitution-of-Kenya)
An AI chatbot utilizing Retrieval-Augmented Generation (RAG) techniques to answer constitutional questions in Kenya, supporting both English and Kiswahili.

### 2. [Lawyer_chatbot](https://github.com/Lord-LLM/AI_Engineering/tree/main/Lawyer_chatbot)
A legal assistance chatbot designed to provide legal information or guidance in a Q&A format, simulating the knowledge and reasoning of a lawyer.

### 3. [SomaAI](https://github.com/Lord-LLM/AI_Engineering/tree/main/SomaAI)
An educational AI application focused on assisting learners, potentially offering tutoring, answering academic questions, or supporting personalized study.

### 4. [Trip_Advisory_Agent](https://github.com/Lord-LLM/AI_Engineering/tree/main/Trip_Advisory_Agent)
An AI-powered trip advisor chatbot, which helps users with travel planning, sightseeing recommendations, and real-time guidance.

### 5. [jac_meal_recommender](https://github.com/Lord-LLM/AI_Engineering/tree/main/jac_meal_recommender)
A personalized meal recommendation system, likely using AI to suggest meals based on user preferences, dietary restrictions, or nutritional needs.

### 6. [mental_health_AI_agent](https://github.com/Lord-LLM/AI_Engineering/tree/main/mental_health_AI_agent)
An AI agent focused on supporting mental health, possibly providing resources, self-care suggestions, or initial conversational assistance for users in need.
